Windows 10 下搭建私人网盘

2020.5.14更新说明

不知道可道云什么时候进行更新了，拥有两个版本方案，KodExplorer（旧，版本号为4.XX） 和 KodBox（新，版本号为1.XX），按自己需求安装组合，注意两个版本的Android端APP不通用。哪个版本是属于哪个APP我也不知道，因为都长一样，不知道官方怎么换版本的。估计很难找到 KodExplorer 版本的APP了，所以推荐安装 KodBox。

  • 方案一：UPUPW + KodExplorer
  • 方案二：UPUPW + KodBox

所需安装包

  1. 根据自己系统需求部署环境UPUPW，选择合适的PHP版本
  2. 下载KodExplorer，
  3. 下载KodBox，环境要求: php 5.3及以上 + mysql/sqlite

安装步骤

安装UPUPW运行环境

  1. 解压缩下载的 UPUPW_NPX.X_XXXX.7z 文件（位置随意）

  2. 然后运行主程序 upupw.exe

  3. 在运行的cmd窗口中输入 s1 回车后启动

  4. 浏览器访问 127.0.0.1，如果显示探针页面则表示启动成功。

  5. 可根据自己需求自定 UPUPW 一些服务， 也可直接使用。

安装可道云

进入 UPUPW_NPX.X\htdocs 目录，删除所有文件，然后将下载的可道云 kodexplorerX.X.zip （或者KodBox）解压到 htdocs 目录

KodExplorer设置（旧）

浏览器访问 127.0.0.1，进入可道云界面，首次进入需要设置 admin 用户的密码

KodBox设置（新）

  • 创建kodbox数据库

    浏览器访问http://localhost/pmd，登录的账号密码都在UPUPW解压目录的【密码及安全说明.txt文件中】，登陆后创建数据库kodbox

  • 初始化设置

    访问http://localhost，选择MySQL数据库，填入上面的账号和密码，数据库名称与上一步创建的相同即可，最后设置管理员账号和密码

局域网的其他设备可通过访问部署机器的局域网地址（192.168.X.X）进入
